# Voice Based Car Unlocking and Controlling System

## 📌 Project Overview
This project is a voice-controlled robotic vehicle that can be operated using speech commands through a smartphone.

## 🚀 Features
- Voice-based control using mobile app
- Bluetooth communication using ESP32
- Motor control using L298N driver
- Security feature (unlock/lock system)
- Obstacle & gas detection using sensors

## 🛠️ Technologies Used
- ESP32 Microcontroller
- Arduino IDE
- Bluetooth Communication
- Embedded C
- Android Application
